# /changes-to-graph

Reads the last commits changes and update codebase knowledge graph.
For this use the /query-graph skill to find the domains to update.
If you cant find a domain to update then you are allowed to read the codebase to understand it better. If after reading the codebase you think that it should create a new domain, then you are allowed to create a new domain.
